几个非常好用的WordPress插件

2011-11-02 · 954 words · 2 minute read
Linux
Linux · WordPress

无论是搭建中小企业的网站，还是创建个人博客，WordPress都可以轻松胜任。除了其本身优秀的设计，无数各种各样为WordPress编写的扩展插件也是大家选择用它的重要原因。

最近为了搭建网站，兔哥花了不少时间，仔细挑选了几个功能扩展、SEO、安全增强方面的WordPress插件。功能强大，超级好用。

功能扩展类 >>>

cforms：联系、问题咨询类提交表单

网址：cforms

兔哥评语：只是简单的配置一下，就可以实现非常专业的提交表单。可以设置自动回复邮件，还可以记录表单提交的数据并提供导出选项。超强，强烈推荐！

Watermark My Image：上传图片时，可以自动增加自己设置的水印

网址：Watermark My Image

兔哥评语：很省心，只是兔哥还没找到上传时可以指定某些图片不加水印的方法。现在碰到这种情况，只好先临时禁用一下该插件，上传完了，再打开。

WP SyntaxHighlighter：程序代码高亮显示，支持多数语言，如Java，PHP，ShellScript,C等

网址：WP SyntaxHighlighter

兔哥评语：类似的插件很多，兔哥感觉这个还不错。

SEO相关 >>>

WordPress SEO：包括了SEO涉及到的各种基本设置项， 包括页面的title，url，内链，外链，meta信息等。

网址：WordPress SEO

兔哥评语：设置很方便，前提是对SEO有一定的了解，或者愿意花时间仔细研究里面的每一项。

Google Analytics for WordPress：让WordPress和Analytics帐号非常方便的对接

网址：Google Analytics for WordPress

兔哥评语：简单易用。

Google XML Sitemaps：自动生成Google sitemap xml的工具

网址：Google XML Sitemaps

兔哥评语：非常实用。可以明显提升在Google上SEO效果。

Ad Injection：AdSense 广告插件

网址：Ad Injection：AdSense

兔哥评语：该插件可以让AdSense广告随机地出现在页面的某个位置上。由于各种广告显示的细节设置非常到位，可以最大限度的让受众不那么讨厌广告。

##安全 >>>

Better WP Security：提升WordPress网站的安全性

网址：Better WP Security

兔哥评语：由于WordPress的用户量巨大，WordPress站点的默认路径结构，默认帐号等没有任何秘密可言，而这这个插件可以很方便的帮助你改变WordPress默认的一些设置，对安全性提高不少。
