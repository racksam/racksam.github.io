电脑清除DNS缓存的方法

2014-04-09 · 283 words · 1 minute read
Mac
DNS

MacOS及Windows不同版本下清除DNS缓存的方法不同。

Mac OS X 10.5, 10.6+

终端窗口下执行以下命令：#sudo dscacheutil -flushcache

Mac OS X 10.7, 10.8, 10.9

终端窗口下执行以下命令：#sudo killall -HUP mDNSResponder

Windows 8

在命令行窗口执行以下命令：ipconfig /flushdns

调出命令行窗口的方法： windows键+R，打开“运行”窗口，然后输入cmd，回车后命令提示符窗口打开。

Windows 7

在命令行窗口执行以下命令：ipconfig /flushdns

调出命令行窗口的方法： 点击开始菜单，在搜索框输入cmd进行搜索， 然后鼠标右键选择以管理员身份运行

Windows XP

在命令行窗口执行以下命令：ipconfig /flushdns

调出命令行窗口的方法： 点击开始菜单，在[运行]框中输入cmd
